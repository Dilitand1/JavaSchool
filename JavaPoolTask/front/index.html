Предварительно:
1. Ставим докер.
2. Запускаем в cmd в папке с проектом "docker-compose up --build"
3. БД поднимется на http://localhost:5435 (логин root, пароль root), рест сервер на http://localhost:8085/myapp
Тестовые запросы:
Метод Назначение Маппинг Параметры Пример (ссылка)
GET Получить весь список машин auto/listAuto без параметров http://localhost:8085/myapp/auto/listAuto
GET Получить машину по id auto/{ID} без параметров http://localhost:8085/myapp/auto/1
GET Получить машину по id auto/getAuto id http://localhost:8085/myapp/auto/getAuto?id=1
GET Удалить машину по id auto/remove id http://localhost:8085/myapp/auto/remove?id=50
POST Добавить машину auto/add без параметров
POST Обновить машину auto/update без параметров
GET Поиск по параметрам auto/listAutoByParams model
minSpeed
maxSpeed
minMileAge
maxMileAge		 http://localhost:8085/myapp/auto/listAutoByParams?model=Rover&maxSpeed=300&minSpeed=200&maxMileAge=300000
